ExamPrep Agent is a Python-based CLI tool designed to streamline academic revision. By leveraging the Gemini LLM, it converts any study topic into a high-yield, structured revision pack tailored to your specific exam type and difficulty level.

## üåü Key Features
* **Contextual Intelligence:** Automatically adapts content for School, University, or Competitive exams (like AFCAT/GRE).
* **Structured Revision Packs:** Generates 60-second snapshots, structured notes, and ASCII architecture diagrams.
* **Active Recall & Testing:** Provides custom-generated Viva/Interview questions and a 10-minute quick quiz for every topic.
* **Memory Anchors:** Includes mnemonics and real-world use cases to improve long-term retention.
* **Spaced Repetition:** Outputs a Day 1, 3, and 7 revision plan based on learning science.
